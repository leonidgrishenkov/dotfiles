# Custom Agents

How to use custom agents.

## `obsidian-writer`

cd into the vault directory and enter TUI:

```sh
cd ~/Filen/Obsidian/main-vault

opencode
```

Then in the TUI run defined commands:

```
@obsidian-writer fix typos in @Base/Git.md
@obsidian-writer write a new note about the Feynman learning technique
@obsidian-writer reformat @Base/Airflow DAG.md
```
